# テストケース

前提条件
- `backend\.env`に
  - 下記が含まれている
    ```
    IHASH_SEED=seed123
    ```
- `backend\src\config\news.json`に
  - 下記が含まれている
    ```
    {
      "info": "荒らし煽りには「無視機能」(荒らし退治も荒らしと同類ですよ)"
    },
    {
      "day": "2022/07/12",
      "info": "お知らせを追加しました"
    }
    ```

重要度: リグレッションテスト観点

- S: 必ず行う
- A: 影響範囲が明確で小さい場合に行う
- B: 影響範囲が明確で大きい場合に行う
- C: 影響範囲が不明確な場合に行う

## スイート "独り言" S

1. http://localhost:2108/ にアクセスする
    - [ ] `入室`画面に遷移すること
1. 名前入力欄に`テストさん#trip`と入力し、`OK`を押下する
    - [ ] `選択`画面に遷移すること
    - [ ] キャラのプレビューに表示されるキャラ情報が`テストさん◆hEpdoZ.tHU`であること
1. `もなちゃと`を押下する
    - [ ] `チャット`画面に遷移すること
    - [ ] 部屋人数が`1`人になっていること
    - [ ] キャラ情報が`テストさん◆hEpdoZ.tHU`であること
1. `こんにちは`と入力し、`OK`を押下する
    - [ ] `こんにちは`と書かれた吹き出しが出現すること

## スイート "チャット画面" B

1. http://localhost:2108/ にアクセスする
1. 名前入力欄に`テストさん#trip`と入力し、`OK`を押下する
1. `もなちゃと`を押下する
1. `こんにちは`と入力し、`Enter`を押下する
    - [ ] 音がでること
    - [ ] 丸い吹き出しが現れること
1. `こんにちは`と入力し、`Shift+Enter`を押下する
    - [ ] 考えごと吹き出しが現れること
1. 反転矢印ボタンを押下する
    - [ ] キャラの向きが変わること
1. 音量セレクタを`OFF`にする
1. `こんにちは`と入力し、`Enter`を押下する
    - [ ] 音がでないこと
1. 時間セレクタを`QUICK`にする
    - [ ] 吹き出しの速度が速くなること
1. 状態セレクタを`ﾏﾀｰﾘ`にする
    - [ ] キャラの状態が`ﾏﾀｰﾘ`になること
1. キャラをドラッグして別の場所にドロップする
    - [ ] 期待踊りの場所に移動すること
